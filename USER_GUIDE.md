# 📡 台股雷達：麻瓜上手指南 (User Guide)

歡迎使用「台股雷達」！這是一份專為非技術人員編寫的設定說明書，帶領您快速啟動監控功能。

---

## 🟢 步驟 1：環境準備 (第一次才需要)

本工具需要在您的電腦上執行 Node.js 程式環境。

1.  **安裝 Node.js**：請點擊 [官方下載連結 (Windows)](https://nodejs.org/dist/v22.14.0/node-v22.14.0-x64.msi) 並安裝。
    - _若未安裝 Node.js，啟動雷達時會報錯消失。_
2.  **確保有 Chrome**：系統會自動調用您電腦中的 Google Chrome 執行爬蟲。

---

## 🟢 步驟 2：快速啟動 (Windows)

在專案資料夾中，找到名為 **「台股公告監控雷達.bat」** 的檔案，點擊兩下即可啟動。

- 程式會自動檢查環境套件。
- 啟動成功後，會自動彈出 Chrome 瀏覽器進入功能介面。
- **請保持黑視窗開啟**：關閉視窗會導致功能停止。

---

## 🟢 步驟 3：自動 Email 通知設定 (選填，非必要)

**重要提醒，這邊不填寫也不會怎樣**

**這功能是做什麼的？**
當雷達抓到您感興趣的公告時，會**立刻寄 Email 給您**。如果您不需要 Email 提醒（只在網頁上看歷史記錄），可以跳過此步。

### A. 申請 Resend API Key (免費)

1. 前往 [Resend 官網 (resend.com)](https://resend.com/) 並註冊帳號。
2. 點擊左側 **「API Keys」** -> **「Create API Key」**。
3. 複製產生的 `re_` 開頭金鑰。

### B. 設定設定檔 (.env)

1. 將專案目錄下的 `.env.example` 複製並更名為 `.env`。
2. 依照以下格式填寫（**或直接複製整段請 AI 幫您填寫**）：

```env
# [必須] 伺服器連接埠 (通常維持 3002)
PORT=3002

# [選填] Resend API 設定 (若不寄信可留空)
RESEND_API_KEY=在此貼上你的_re_開頭金鑰
EMAIL_FROM=在此填寫你註冊Resend的Email

# [選填] Chrome 路徑 (若程式自動偵測失敗才需要填寫)
# CHROME_PATH=C:\Program Files\Google\Chrome\Application\chrome.exe
```

---

## 🟢 步驟 4：測試與使用

1. 啟動程式後，在網頁上的「通知設定」填入您的收件 Email。
2. 點擊 **「儲存設定」**。
3. 點擊 **「測試寄信」**，確認您的信箱能收到系統發出的測試信。
   - _注意：Resend 免費版在未驗證網域前，收件人必須是你註冊用的那個 Email。_

---

## 🔴 常見問題排除 (Troubleshooting)

- **錯誤：點開 .bat 閃退** -> 通常是沒有安裝步驟 1 的 Node.js。
- **錯誤：找不到 Chrome** -> 確保您有安裝 Google Chrome。若裝在非預設位置，請在 `.env` 手動指定 `CHROME_PATH`。
- **重設設定**：如果您想清空所有歷史記錄，刪除 `data/mops.db` 檔案即可。
