📖 如何設定自動 Email 通知

1. 註冊 Resend 帳號

前往 resend.com 並註冊一個免費帳號。

2. 取得 API Key

登入後點擊左側 "API Keys" -> "Create API Key"，複製產生的 re_ 開頭代碼。

3. 填入設定 (.env)

啟動器已自動在程式資料夾建立 .env，用記事本開啟後填入：

RESEND_API_KEY=貼上你的金鑰
EMAIL_FROM=註冊Resend的Email

填完後存檔，重新啟動程式即可生效。

⚠️ 重要：Resend 免費版在驗證網域前，收件人必須與註冊帳號相同。設定完成後請重啟程式。
